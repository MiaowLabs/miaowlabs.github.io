概述

这是喵呜实验室（Miaow labs）两轮自平衡小车项目（代号ISO）的配套书籍。

本书适合所有对两轮自平衡小车感兴趣的读者阅读。

两轮自平衡小车是集成机械/电子/通信/自动化/嵌入式等技术于一体的小型智能机器人。通过制作两轮自平衡小车，读者能大幅度增强以上提到的技术能力。

例如，自动控制理论是自动化专业学生的一门必修专业基础课，课程中的一些概念相对比较抽象，如系统的稳定性、可控性、收敛速度和抗干扰能力等。两轮自平衡小车是一个典型的非线性、强耦合、多变量和不稳定系统，作为控制系统的被控对象，它是一个理想的教学实验设备，许多抽象的控制概念都可以通过两轮自平衡小车直观地表现出来。

如果你想DIY两轮自平衡小车，可以根据本书籍制作出属于自己的ISO两轮自平衡小车。

你会发现喵呜ISO两轮自平衡小车是最适合新手入门的开源两轮自平衡小车平台。

参与项目

有些朋友可能不太清楚如何加入项目，我这里写一个简单的流程，大家可以参考一下：

  1. 首先fork我的项目
  2. 把fork过去的项目也就是你的项目clone到你的本地
  3. 在命令运行行 git branch develop 來创建一个新分支
  4. 运行 git checkout develop 來切换到新分支
  5. 运行 git remote add upstream https://github.com/tommy60703/the-swift-programming-language-in-traditional-chinese 把我的库添加为远端库
  6. 运行 git remote update更新
  7. 运行 git fetch upstream master 拉取我的库的更新到本地
  8. 运行 git rebase upstream/master 将我的更新合并到你的分支

这是一个初始化流程，只需要做一遍就行，之后请一直在 develop 分支进行修改。

如果修改过程中我的库有了更新，请重复 6、7、8 步。

修改之后，首先push到你的库，然后登陆Github，在你的repo的首页可以看到一个pull request按键，点击它，填写一些说明资讯，然后提交即可。
