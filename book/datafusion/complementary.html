开源造车—喵呜ISO两轮自平衡小车

互补滤波

流传得最广泛的是MIT（麻省理工）学生Shane Colton的于2007年发表的一篇论文《The Balance Filter》。Shane Colton在其论文中提到，对于集成陀螺仪与加速度传感器的平衡装置，互补滤波是一种简单有效的解决方案。

《The Balance Filter》里面提到的就是一阶互补滤波。

一阶互补滤波核心代码非常简洁：

angle = (0.98)*(angle + gyro * dt) + (0.02)*(x_acc);

其中，angle为得到的实际角度，gyro为陀螺仪值，x_acc为加速度值，dt为计算周期；

后来经过众人的不断改善，又发展出了滑动参数互补滤波和二阶互补滤波等等。

附上二阶滤波代码供大家参考：

二阶互补滤波

// newAngle = angle measured with atan2 using the accelerometer
// newRate = angle measured using the gyro
// looptime = loop time in millis()
float Complementary2(float newAngle, float newRate, intlooptime)
{
float k = 10;
float dtc2 = float(looptime) / 1000.0;
x1 = (newAngle -   x_angle2C) * k * k;
    y1 = dtc2 * x1 + y1;
x2 = y1 + (newAngle -   x_angle2C) * 2 * k + newRate;
    x_angle2C = dtc2 * x2 + x_angle2C;
return x_angle2C;
}
